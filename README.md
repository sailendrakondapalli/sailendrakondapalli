# Sailendra Kondapalli - Portfolio Website

A cinematic, high-end personal portfolio website built with React + Vite, featuring smooth animations, glassmorphism design, and premium typography.

## 🚀 Features

- **Cinematic Design**: Dark gradient background with animated particles
- **Smooth Animations**: Framer Motion powered transitions and reveals
- **Custom Cursor**: Interactive cursor effect
- **Responsive Design**: Fully responsive across all devices
- **Premium UI**: Glassmorphism cards and magnetic buttons
- **Project Showcase**: Detailed project pages with galleries
- **Contact Form**: Integrated contact form with WhatsApp button

## 📦 Installation

1. Install dependencies:
```bash
npm install
```

2. Start the development server:
```bash
npm run dev
```

3. Build for production:
```bash
npm run build
```

4. Preview production build:
```bash
npm run preview
```

## 🎨 Sections

- **Hero**: Full-screen hero with animated typing effect
- **About**: Personal introduction with animated stats
- **Skills**: Interactive skill bars with categories
- **Projects**: Portfolio showcase with hover effects
- **Testimonials**: Client reviews
- **Contact**: Contact form and direct communication options
